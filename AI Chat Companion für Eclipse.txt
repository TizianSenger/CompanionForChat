🧠 Konzept: AI Chat Companion für Eclipse

Ein visueller 3D-Companion in Eclipse, der:

4️⃣ Companion State Machine (zentral!)

Der Companion denkt nicht selbst, sondern visualisiert Zustände.

Beispiel-States
IDLE
THINKING
RESPONDING
HAPPY
CONFUSED
ERROR
TIRED
FOCUSED
Beispiel-Mapping
State	Bedeutung	Animation
IDLE	nichts los	Idle / Breathing
THINKING	LLM arbeitet	Kopf senken / nachdenken
RESPONDING	Antwort kommt	Sprechen / Nicken
ERROR	Compiler / LLM Fehler	Kopf schütteln
HAPPY	Erfolg	Freuen / Winken
TIRED	lange Session	Gähnen / Dehnen

➡️ kein Chatbot im Avatar
➡️ reiner Emotions-Renderer

5️⃣ Verbindung zu deinem bestehenden LLM-Plugin
Java-Seite

Dein LLM-Plugin feuert Events wie:

onLLMRequestStarted

onLLMResponseReceived

onLLMError

Diese Events gehen an den Companion Controller.

Übergabe an WebView
browser.execute("companion.setState('THINKING')");
browser.execute("companion.setState('RESPONDING')");
browser.execute("companion.setState('ERROR')");

➡️ keine enge Kopplung
➡️ Companion ist ein „Subscriber“
